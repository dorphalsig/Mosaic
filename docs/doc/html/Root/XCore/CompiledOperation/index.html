CompiledOperation

Overview

CompiledOperation is the type of all XMF compiled operations. A compiled operation can be invoked using 'invoke/2' or by applying it to its arguments. A compiled operation consists of machine code instructions. A compiled operation may be associated with its source code to aid debugging.

@Operation removeNameSpace(n : NameSpace):XCore::Element
  let findSeq = self.dynamics()->asSeq;
      findDone = false;
      findResult = null
  in declare(hidden,findSeq);
     declare(hidden,findDone);
     declare(hidden,findResult);
     @While not findSeq->isEmpty and not findDone do
       let pair = findSeq->head then
           findElement = findSeq->head
       in declare(hidden,findElement);
          if pair->head = 2 and pair->tail = n.contents
          then 
            findDone := true;
            findResult := self.setDynamics(self.dynamics()->excluding(pair,false))
          else findSeq := findSeq->tail
          end 
       end
     end;
     if not findDone
     then false
     else findResult
     end 
  end
end
