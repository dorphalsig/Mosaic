ModelBrowserCommandInterpreter

Overview

This class defines a command interpreter over model browsers. It can be used as the basis for defining event driven mappings. The mapping handles the following output commands by sen-ding them in the appropriate format to the client connected to the output channel: [newModelBrowserCommandCommand(modelBrowser)]
