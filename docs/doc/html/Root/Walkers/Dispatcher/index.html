Dispatcher

Overview

A dispatcher, like a walker, passes over an element performing a task. The difference between a dispatcher and a walker is that the dispatcher can be programmed on a type-by-type basis and then dispatches to handlers whereas the types in a walker are fixed. A walker may be slightly more efficient, but less extensible and modular.
