DefinitionManager The XTools definition manager is the container for all the definitions created using XTools so they are all in one place.
ObjectManager The XTools object manager is the container for all the objects created using XTools so they are all in one place. Menus are added to ObjectManager to enable new elements to be added.
SavedToolsNo Documentation Specified
Tool A tool provides an interface over an element. The interface may be realised in a number of different ways (for example a diagram or a form). Changes to the interface cause events to be raised and handled by an element manager. Changes to the element and its sub-components (may) cause events to be raised and handled by the element manager.
ToolDescriptor A tool descriptor associates a tool type with an element manager. Tool descriptors are added to classifiers and are used to produce language specific tools such as property editors and diagram editors.
ToolManager A tool manager maintains associations between classes and a collection of XTool descriptors. Each tool descriptor manages a collection of tools.
ToolType A tool type defines the type of a tool. Each tool uses the tool type to resolve named references to component types.
