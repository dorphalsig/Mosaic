Tool

Overview

A tool provides an interface over an element. The interface may be realised in a number of different ways (for example a diagram or a form). Changes to the interface cause events to be raised and handled by an element manager. Changes to the element and its sub-components (may) cause events to be raised and handled by the element manager.
