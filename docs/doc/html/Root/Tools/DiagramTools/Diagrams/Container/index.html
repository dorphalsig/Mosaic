Container

Overview

A container has a sequence of display element types and a sequence of display elements (essentially a node or a box). Each type in the container type is named as is the contents. Therefore elements can be referenced by supplying a path from the receiver to a referenced element. A slight complication occurs when referenced elements occur as part of a STAR-ed type. This means that references produce sequences of elements.
