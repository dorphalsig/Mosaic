Formatted

Overview

An instance of this class is a type definition for a container of display elements. In an instanc of the type, the display elements are formatted using a layout manager. The layout manager can use the value of the enumerated type layout to control how the contained elements are displayed. The value of layout should be one of: VERTICAL for layout along the y-axis; HORIZONTAL for layout along the x-axis; OVERLAY for elements on top of each other.
