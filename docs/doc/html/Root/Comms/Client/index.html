Client

Overview

A class that implements a client interface. Use this class to connect to a server either in blocking or non-blocking mode. By default, a client is blocking.

connect(address:String,port:Integer):Boolean Supply an address string and a port number to attempt to connect to the server at the given address and port. If the client is blocking then the call will only return when the connection is successful; meanwhile other threads are given a chance to run. If the client is non blocking then it will return immediately with true or false depen-ding on whether the connection was successful.
