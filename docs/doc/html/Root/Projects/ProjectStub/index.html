ProjectStub

Overview

A project stub is used as a placeholder for a project that exists but is not loaded. The manifest of the stub describes how to build and load the project.

  in if owner.isReallyKindOf(ProjectManager)
     then if package.isReallyKindOf(XCore::Package)
          then let projectManager = owner
               in owner.remove(self);
                  projectManager.add(Project(package,resourceName,manifest))
               end

          end

  end
