Projects

Overview

This package defines classes that implement XMF projects.


Packages


Classes
DiagramNo Documentation Specified
ProjectNo Documentation Specified
ProjectManager A project manager has a collection of projects. The manager is associated with a resource name that defines the default location for the projects. The resource associated with the project manager also contains a manifest file that lists all of the projects.
ProjectManagerContainer A project manager container has a collection of project managers. This is distinct from a project manager in that it is not inten-ded to contain projects directly. Currently this is enforced only by a more restricted set of menus for user interaction
ProjectStub A project stub is used as a placeholder for a project that exists but is not loaded. The manifest of the stub describes how to build and load the project.
